PDF to Speech Converter

This Python project converts PDF files to speech audio. It can be used to create audiobooks, podcasts, or other audio content from text. I am particularly proud of this project because it solves a real-world problem for many people, such as those with visual impairments.

Breakout Game

This Python project is a classic Breakout Game developed using the Tkinter library. You can break bricks and score points while testing your reflexes and having fun!

Tic-Tac-Toe

This project is a text-based implementation of the classic Tic-Tac-Toe game in Python. You can play against another player or against an AI opponent. Challenge your friends and see who wins!
