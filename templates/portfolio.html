Morse Code Converter

Morse Code Converter is a Python project I developed. It allows you to convert text messages into Morse code and vice versa. It's a simple yet fun project that demonstrates my coding skills and creativity.

The project takes user input and utilizes a dictionary to translate each character into its corresponding Morse code representation. It also provides a sound option to hear the Morse code audio.

Technologies used: Python
